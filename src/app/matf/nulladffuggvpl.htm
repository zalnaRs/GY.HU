A nulladfokú (konstans) függvény olyan lineáris függvény, ahol az

 

f(x) = a*x + b      a, b Î R

 

képletben a = 0.

 

Így a képlete:

 

f(x) =  b     

 

   A konstans (állandó) függvény az értelmezési tartományának minden eleméhez ugyanazt a számot rendeli. A konstans függvény grafikonja mindig az x tengellyel párhuzamos egyenes.

 

 

Példa

 

Konstans függvény

 

 

 

 

Az x  4 képlettel megadott konstans függvény grafikonja.

 

 

 

 

 

 

 

 

 

Az x  0 képlettel megadott konstans függvény képe pontosan az x tengely.
