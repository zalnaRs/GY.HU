A derékszögû háromszög befogóira emelt négyzetek területének az összege egyenlõ az átfogóra emelt négyzet területével.

 

Bizonyítandó:

a2 + b2 = c2

 

 

 

 

Bizonyítás:

 

   Az I. ábrán az ABC∆ befogói: a, b.

   A II. ábrán az a+b oldalú négyzetet felosztottuk négy, az ABC∆-gel egybevágó háromszögre, (A két befogó és a közbezárt szög megegyezik.) és egy c oldalú rombuszra.

Belátható:

(a + b) + g = 180°, mert egyenes szög.

 

   Felhasználva, hogy a derékszögû háromszög hegyesszögeinek összege 90°, (a + b) helyére behelyettesítve a 90°-ot:

 

      90° + g = 180°   // - 90°

g = 90°

T¯  = c2

 

 

   A II. ábrán az a+b oldalú négyzet területe:

 

T£ = (a + b)2

T£  4T∆ = c2

a2 + b2 = c2
