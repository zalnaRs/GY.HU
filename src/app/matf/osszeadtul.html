1.

 

 

Az összeadandók sorrendje tetszés szerint felcserélhetõ, az összeg nem változik. (kommutatív tulajdonság)

 

a + b = b + a

a + b + c + d = b + a + d + c

 
 
 

 

Példa

 
 
 

a.       5 + 7 = 12           7+ 5 = 12

 
 
 
 
 
 

b.       4 + 3 + 5 + 2 = 14        3 + 4 + 2 + 5 = 14        5 + 4 + 2 + 3 = 14

 
 
 

2.

 
 
 

 

Az összeadandók tetszés szerint csoportosíthatók, - zárójelezhetõk -, az összeg nem változik. A zárójel el is hagyható. (asszociatív tulajdonság)

 

a + (b + c) = (a + b) + c

(a + b) + c = a + b + c

a + (b + c + d) =  a + (b + c) + d

 
 
 
 

Példa

 
 
 

a.     5 + (7 + 3) = 15          (5 + 7) + 3  = 15

 
 

b.     4 + (3 + 5 + 2) = 14          4 + (3 + 5) + 2 = 14          (4 + 3 + 5) + 2 = 14
