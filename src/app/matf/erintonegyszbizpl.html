   Bizonyítandó:

 

AB + CD = AD + BC

 

   Az ábrán azonos színnel, és azonos betûvel jelöltük a közös külsõ pontból kiinduló érintõszakaszokat. Ezek a szakaszok egyenlõ hosszúak.

 

AB + CD = (a + b) + (c + d) = a + b + c + d

 

AD + BC = (a + d) + (b + c) = a + d + b + c

 

   Az összeadásnál a zárójel elhagyható. A két összeg csak a tagok sorrendjében különbözik egymástól. Összeadásnál a tagok sorrendje tetszés szerint felcserélhetõ, az összeg nem változik.

 

   Ezt felhasználva:

 

AB + CD = AD + BC
