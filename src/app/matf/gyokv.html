A gyökvonás mûvelete a hatványozás egyik megfordítása. A hatványozásban szereplõ mennyiségek a hatványalap, hatványkitevõ és a hatványérték. Hatványozásnál az alap és a kitevõ ismeretében számoljuk a hiányzó hatványértéket.

 

 

 

 

 

 

 

 

25 = a

 

a = 2*2*2*2*2 = 32

 

 

 

 

 

 

 

34 = b

 

b = 3*3*3*3 = 81

 

 

 

 

 

 

 

72 = c

 

c = 7*7 = 49

 

 

 

 

 

 

 

53 = d

 

d = 5*5*5 = 125

 

 

 

 

 

 

 

 

 

 

 

 

 

 

  

   A gyökvonás az a mûvelet, amikor a kitevõ és a hatványérték ismeretében keressük az ismeretlen hatványalapot.
